Detailed Description

PCC1 Command data structure definition.

The command data structure contains the information to run a command in the PPC1 controller. The final objective is to define a class of commands able to control the PPC1 controller and run a macro as a set of commands.

Usage:

  • define the object : fluicell::PPC1api::command *my_command;
    • a macro is a vector of commands : std::vector<fluicell::PPC1api::command> *_macro;
    • run the command still missing implementation !!
